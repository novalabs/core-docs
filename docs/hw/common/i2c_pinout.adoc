|================
|Pin # |Name    |Description
|1     |VCC     |Power supply (3.3V)
|2     |GND     |Ground
|3     |SCL     |Clock
|4     |SDA     |Data
|================

The I2C port connector is a 4-pin 1.25mm Molex Picoblade (PN 53047-0410, mating connector PN 51021-0410).
